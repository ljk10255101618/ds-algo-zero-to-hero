# 学习数据结构与算法

## 📚 项目简介

这是一个从零开始学习数据结构与算法的完整学习路径，使用费曼学习法，将复杂的技术概念转化为易懂的教学内容。

## 🎯 学习目标

- 理解常见数据结构的核心原理
- 掌握基本算法思想
- 建立算法思维框架
- 能够分析算法复杂度
- 解决实际编程问题

## 🌟 特色

- **费曼学习法**：12岁小孩也能懂的解释
- **问题导向**：每个概念都说明解决了什么问题
- **渐进式学习**：从简单到复杂，平滑过渡
- **互动验证**：分级思考题和参考答案
- **代码示例**：完整可运行的 Python 代码
- **复杂度分析**：每种数据结构/算法都有性能分析

## 🛠️ 使用方法

1. 按照目录顺序学习
2. 每个文档都包含思考题，建议完成后再继续
3. 代码示例建议自己动手敲一遍

## 📝 学习原则

1. **12岁小孩也能懂**：使用日常生活类比
2. **拆成原子知识点**：每个概念都拆到最基础
3. **不出现陌生词汇**：新术语立即解释
4. **拒绝知识诅咒**：不假设任何背景知识
5. **用提问引导**：每节都以问题开始
6. **类比优先**：先类比再技术解释
7. **视觉化**：使用图表、代码块
8. **递进式**：从简单到复杂
9. **互动式**：包含思考题和答案

## 🧠 核心思维框架

```
遇到问题 → 选择数据结构 → 选择算法 → 分析复杂度 → 优化
```

## 📊 复杂度速查表

| 复杂度 | 数组(查) | 链表(查) | 栈/队列 | 哈希表 | 二叉树 |
|--------|---------|---------|---------|--------|--------|
| 访问   | O(1)    | O(n)    | O(n)    | O(n)   | O(n)   |
| 查找   | O(n)    | O(n)    | O(n)    | O(1)*  | O(n)   |
| 插入   | O(n)    | O(1)    | O(1)    | O(1)*  | O(n)   |
| 删除   | O(n)    | O(1)    | O(1)    | O(1)*  | O(n)   |

*平均复杂度，最坏情况 O(n)
