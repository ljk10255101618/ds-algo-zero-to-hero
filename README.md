# DS-Algo Zero to Hero - 数据结构与算法从零到精通

> 🎯 **DSATutor** - 可视化交互学习平台 · 建立直觉 · 理解本质 · 掌握算法

## 📚 项目简介

这是一个从零开始学习数据结构与算法的完整学习路径，使用费曼学习法，将复杂的技术概念转化为易懂的教学内容。

## 🎯 学习目标

- 理解常见数据结构的核心原理
- 掌握基本算法思想
- 建立算法思维框架
- 能够分析算法复杂度
- 解决实际编程问题

## 🧠 可视化教学平台

访问 [DSATutor 主页](./index.html) 体验交互式学习：

- **🧠 知识图谱**：D3.js 力导向图展示27个知识节点的关系网络
- **📚 快速入口**：按类别（线性结构/树形结构/图/排序/搜索/高级结构）和难度（Lv1-Lv5）组织
- **🤖 AI学习助手**：内置知识库的智能问答，支持12+主题
- **🎮 交互可视化**：数组操作、二叉树遍历、快速排序等动画演示
