## 二、基础查找

### 2.1 线性查找（Linear Search）

**思想**：从头到尾逐个检查

```python
def linear_search(arr, target):
    """线性查找"""
    for i, val in enumerate(arr):
        if val == target:
            return i
    return -1


def linear_search_all(arr, target):
    """查找所有匹配的索引"""
    return [i for i, val in enumerate(arr) if val == target]


# 测试
arr = [64, 25, 12, 22, 11, 90, 3]
print(linear_search(arr, 22))     # 3
print(linear_search(arr, 100))  # -1 (未找到)
```

**时间复杂度**：O(n)

### 2.2 二分查找（Binary Search）

**思想**：每次排除一半，缩小搜索范围

```
猜数字游戏：1-100 之间猜数字

第一次猜：50
比50小？→ 排除 50-100，只剩 1-49
第二次猜：25
比25小？→ 排除 25-49，只剩 1-24
第三次猜：12
比12大？→ 排除 1-12，只剩 13-24
第四次猜：18
猜对了！

每猜一次，数据量减半！
100 个数最多 7 次猜完（log₂ 100 ≈ 7）
```

```python
def binary_search(arr, target):
    """
    二分查找（循环版本）
    前提：数组已排序
    """
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1


def binary_search_recursive(arr, target, left=None, right=None):
    """二分查找（递归版本）"""
    if left is None:
        left, right = 0, len(arr) - 1
    
    if left > right:
        return -1
    
    mid = (left + right) // 2
    
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return binary_search_recursive(arr, target, mid + 1, right)
    else:
        return binary_search_recursive(arr, target, left, mid - 1)


# 测试
arr = [11, 12, 22, 25, 34, 64, 90]
print(binary_search(arr, 25))     # 3
print(binary_search(arr, 100))   # -1
```

### 2.3 二分查找变体

```python
def lower_bound(arr, target):
    """
    查找第一个 >= target 的元素索引
    如果所有元素都 < target，返回 len(arr)
    """
    left, right = 0, len(arr)
    
    while left < right:
        mid = (left + right) // 2
        if arr[mid] < target:
            left = mid + 1
        else:
            right = mid
    
    return left


def upper_bound(arr, target):
    """
    查找第一个 > target 的元素索引
    如果所有元素都 <= target，返回 len(arr)
    """
    left, right = 0, len(arr)
    
    while left < right:
        mid = (left + right) // 2
        if arr[mid] <= target:
            left = mid + 1
        else:
            right = mid
    
    return left


def binary_search_leftmost(arr, target):
    """
    查找第一个等于 target 的元素索引
    二分查找找到的不一定是第一个
    """
    left, right = 0, len(arr) - 1
    result = -1
    
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            result = mid
            right = mid - 1  # 继续向左找
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return result


# 测试
arr = [1, 2, 2, 2, 3, 4, 5]
print(lower_bound(arr, 2))      # 1（第一个 >= 2）
print(upper_bound(arr, 2))      # 4（第一个 > 2）
print(binary_search_leftmost(arr, 2))  # 1（第一个 == 2）
```

## 三、高级查找

### 3.1 插值查找（Interpolation Search）

**思想**：利用数据分布规律，猜测目标位置

```
适用场景：数据分布均匀
例如：[1, 10, 20, 30, 40, 50]均匀分布

公式：pos = left + ((target - arr[left]) * (right - left) // (arr[right] - arr[left]))

猜 30：pos = 0 + ((30-1) * 5 // 49) = 2
直接命中！
```

```python
def interpolation_search(arr, target):
    """
    插值查找
    适用于数据分布均匀的情况
    """
    left, right = 0, len(arr) - 1
    
    while left <= right and target >= arr[left] and target <= arr[right]:
        if left == right:
            if arr[left] == target:
                return left
            return -1
        
        # 插值公式
        if arr[right] == arr[left]:
            break
        pos = left + int(
            ((target - arr[left]) * (right - left)) / 
            (arr[right] - arr[left])
        )
        
        if arr[pos] == target:
            return pos
        elif arr[pos] < target:
            left = pos + 1
        else:
            right = pos - 1
    
    return -1


# 测试
arr = [1, 10, 20, 30, 40, 50, 60]
print(interpolation_search(arr, 30))  # 3
```

**时间复杂度**：O(log log n)（数据均匀），最坏 O(n)

### 3.2 指数查找（Exponential Search）

**思想**：先定位目标所在范围，再二分查找

```
步骤：
1. 找到 target 所在范围：2^0 到 2^1，2^1 到 2^2，...
2. 在找到的范围内二分查找
```

```python
def exponential_search(arr, target):
    """
    指数查找
    适用于无界数组或目标在数组开头的情况
    """
    if arr[0] == target:
        return 0
    
    # 找到目标所在范围
    i = 1
    while i < len(arr) and arr[i] < target:
        i *= 2
    
    # 在 [i/2, min(i, n)] 范围内二分查找
    return binary_search_range(arr, target, i // 2, min(i, len(arr) - 1))


def binary_search_range(arr, target, left, right):
    """在指定范围内二分查找"""
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1


# 测试
arr = [1, 2, 4, 8, 16, 32, 64, 128, 256]
print(exponential_search(arr, 32))   # 5
print(exponential_search(arr, 7))   # -1
```

## 四、哈希查找

### 4.1 基于哈希表的查找

```python
class HashTable:
    """简单的哈希表实现"""
    
    def __init__(self, size=10):
        self.size = size
        self.table = [[] for _ in range(size)]
    
    def _hash(self, key):
        return key % self.size
    
    def put(self, key, value):
        idx = self._hash(key)
        for i, (k, v) in enumerate(self.table[idx]):
            if k == key:
                self.table[idx][i] = (key, value)
                return
        self.table[idx].append((key, value))
    
    def get(self, key):
        idx = self._hash(key)
        for k, v in self.table[idx]:
            if k == key:
                return v
        return None
    
    def __contains__(self, key):
        return self.get(key) is not None


# 测试
ht = HashTable()
ht.put(10, "apple")
ht.put(20, "banana")
ht.put(30, "cherry")

print(ht.get(20))      # banana
print(10 in ht)        # True
print(15 in ht)        # False
```

**时间复杂度**：O(1) 平均，最坏 O(n)

## 五、树形查找

### 5.1 二叉搜索树查找

```python
class BSTNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None


def bst_search(root, target):
    """BST 查找"""
    while root:
        if root.val == target:
            return root
        elif target < root.val:
            root = root.left
        else:
            root = root.right
    return None


def bst_range_search(root, low, high):
    """BST 范围查找：找 [low, high] 区间内的所有元素"""
    result = []
    
    def inorder(node):
        if not node:
            return
        if low <= node.val <= high:
            result.append(node.val)
        if node.val > low:
            inorder(node.left)
        if node.val < high:
            inorder(node.right)
    
    inorder(root)
    return result


# 测试
#       8
#      / \
#     3   10
#    / \    \
#   1   6    14

root = BSTNode(8)
root.left = BSTNode(3)
root.right = BSTNode(10)
root.left.left = BSTNode(1)
root.left.right = BSTNode(6)
root.right.right = BSTNode(14)

print(bst_search(root, 6))    # BSTNode(6)
print(bst_range_search(root, 3, 10))  # [1, 3, 6, 8, 10]
```

### 5.2 平衡二叉树（AVL / 红黑树）

```
BST 的问题：可能退化成链表
  查找：从 O(log n) 退化成 O(n)

解决：平衡二叉树
  - AVL：严格平衡，查找 O(log n)
  - 红黑树：近似平衡，插入删除更快

Python 的 dict/set：用的是哈希表
Java 的 TreeMap：用的是红黑树
```

## 六、分块查找

### 6.1 思想

```
适用场景：数据量大且不便一次性加载到内存

思路：
1. 把数据分成若干块
2. 建立块索引（每块的最大值）
3. 先在索引中找到目标可能所在的块
4. 在该块内线性查找

示例：学生按分数分块
块1: 0-59  (最大值59)
块2: 60-79 (最大值79)
块3: 80-100(最大值100)

查78分：
先查索引：78 < 79，在块2
在块2内线性查找
```

```python
def block_search(arr, target, block_size=5):
    """
    分块查找
    arr: 已按块有序排列（块内无序，块间有序）
    """
    n = len(arr)
    
    # 建立块索引：每块的最大值
    block_max = []
    for i in range(0, n, block_size):
        block_max.append(max(arr[i:i + block_size]))
    
    # 找到目标可能所在的块
    block_idx = 0
    while block_idx < len(block_max) and block_max[block_idx] < target:
        block_idx += 1
    
    if block_idx >= len(block_max):
        return -1
    
    # 在该块内线性查找
    start = block_idx * block_size
    end = min(start + block_size, n)
    for i in range(start, end):
        if arr[i] == target:
            return i
    
    return -1


# 测试
arr = [7, 15, 23, 45, 51, 62, 73, 89, 95, 100]
print(block_search(arr, 73))   # 6
print(block_search(arr, 50))   # -1
```

**时间复杂度**：O(√n)

## 七、知识总结

### 7.1 一句话总结

```
线性查找：挨个找，简单但慢
二分查找：折半找，前提要排序
插值查找：猜位置，数据要均匀
指数查找：先定范围，再二分
哈希查找：O(1)，用空间换时间
树形查找：O(log n)，动态数据
分块查找：折中方案，大数据用
```

### 7.2 复杂度速查

| 查找算法 | 时间复杂度 | 空间复杂度 | 前提条件 |
|---------|-----------|-----------|---------|
| 线性查找 | O(n) | O(1) | 无 |
| 二分查找 | O(log n) | O(1) | 有序数组 |
| 插值查找 | O(log log n) | O(1) | 数据均匀 |
| 指数查找 | O(log n) | O(log n) | 有序数组 |
| 哈希查找 | O(1) 平均 | O(n) | 哈希函数 |
| BST查找 | O(log n) 平均 | O(log n) | BST |
| 分块查找 | O(√n) | O(√n) | 块有序 |

### 7.3 选择指南

```
有序数组查特定值：二分查找 ✅
数据均匀分布：插值查找 ✅
无序数据：哈希查找 ✅
动态数据（频繁插入删除）：BST ✅
海量数据分块：分块查找 ✅
```

## 八、自我检验

### 思考题

1. **基础题**：二分查找为什么比线性查找快？
2. **进阶题**：如何在有序数组中找出绝对值最小的元素？
3. **应用题**：如何在 100 亿个整数中快速查找某个数？
4. **思考题**：为什么很多系统用 B+ 树而不是哈希表做索引？

### 参考答案

1. **二分查找快**：每次排除一半，指数级减少搜索范围。100个数，线性最多100次，二分最多7次。

2. **找绝对值最小的元素**：
   ```python
   def find_min_abs(arr):
       if arr[0] >= 0:
           return arr[0]  # 全正，第一个
       if arr[-1] <= 0:
           return arr[-1]  # 全负，最后一个
       # 有正有负，二分找正负分界
       # 正数中找最小的，负数中找绝对值最小的
   ```

3. **100亿整数查找**：
   - 100 亿整数 ≈ 40GB
   - 分块加载到内存（如每块 4MB）
   - 块索引用 B+ 树或哈希
   - 或者用位图/布隆过滤器快速判断存在

4. **B+ 树 vs 哈希表**：
   - B+ 树支持范围查询（>、<、BETWEEN）
   - B+ 树支持排序
   - B+ 树支持高效遍历
   - 哈希表只支持等值查询
