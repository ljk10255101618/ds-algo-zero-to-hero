# DS-Algo OJ 系统使用指南

## 🎯 简介

这是一个轻量级的在线评测系统（Online Judge），用于练习数据结构与算法的题目。

## 📁 目录结构

```
oj/
├── config.json              # 系统配置
├── README.md                # 本文档
├── problems/                # 题目目录
│   ├── 1001_two_sum.json    # 两数之和
│   ├── 1002_merge_two_lists.json
│   ├── 2001_valid_parentheses.json
│   └── ...（更多题目）
├── judge/                   # 评测脚本
│   └── judge.py            # Python 评测器
├── submissions/            # 提交记录
│   └── submission_*.json   # 每次提交的结果
└── web/                    # 前端界面
    └── index.html          # 可视化界面
```

## 🚀 快速开始

### 方式一：命令行评测

```bash
cd oj

# 方式1：指定代码文件
python judge/judge.py 1001 solution.py

# 方式2：交互式输入
python judge/judge.py 1001
# 然后粘贴代码，输入 END 结束

# 方式3：查看所有题目
python judge/judge.py
```

### 方式二：网页界面

```bash
# 直接用浏览器打开
open web/index.html

# 或者用 Python 启动本地服务器
cd web
python -m http.server 8080
# 然后访问 http://localhost:8080
```

## 🔧 添加新题目

1. 在 `problems/` 目录下创建新的 JSON 文件
2. 命名格式：`{题号}_{题目标识}.json`

```json
{
  "id": 1001,
  "title": "题目名称",
  "difficulty": "简单/中等/困难",
  "tags": ["标签1", "标签2"],
  "chapter": "对应章节",
  "description": "题目描述",
  "examples": [...],
  "constraints": [...],
  "template": {...},
  "test_cases": [...],
  "hints": [...],
  "solutions": {...}
}
```

## ⚠️ 注意事项

1. **本地评测**：命令行评测需要 Python 3.8+
2. **测试用例**：每道题至少包含 5 个测试用例
3. **提交记录**：每次提交会自动保存到 `submissions/` 目录
4. **网页界面**：网页端目前只支持查看题目，评测需要使用命令行

## 🎓 学习路径

建议按以下顺序刷题：

1. **入门** → 两数之和、合并链表、有效括号、爬楼梯
2. **进阶** → 无重复字符的最长子串、岛屿数量、合并区间
3. **深入** → 搜索旋转数组、电话号码组合、分发饼干
