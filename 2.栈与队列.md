## 二、栈（Stack）

### 2.1 栈的定义

**专业解释**：栈是一种特殊的线性表，只允许在一端（栈顶）进行插入和删除操作。

```
栈的结构：
        ←── 栈顶（top）
    ┌─────────┐
    │    5     │  ← push(5) 入栈
    ├─────────┤
    │    3     │  ← push(3) 入栈
    ├─────────┤
    │    1     │  ← push(1) 入栈
    ├─────────┤
    │    ...   │
    └─────────┘
        ←── 栈底（bottom）
```

### 2.2 栈的基本操作

| 操作 | 说明 | 时间复杂度 |
|------|------|-----------|
| push(x) | 入栈，向栈顶添加元素 | O(1) |
| pop() | 出栈，移除栈顶元素 | O(1) |
| top() / peek() | 查看栈顶元素 | O(1) |
| isEmpty() | 判断是否为空 | O(1) |
| size() | 返回栈的大小 | O(1) |

### 2.3 栈的代码实现

```python
class Stack:
    """基于 Python list 的栈实现"""
    
    def __init__(self):
        self._items = []
    
    def push(self, item):
        """入栈"""
        self._items.append(item)
    
    def pop(self):
        """出栈"""
        if self.is_empty():
            raise IndexError("Stack is empty")
        return self._items.pop()
    
    def top(self):
        """查看栈顶元素"""
        if self.is_empty():
            raise IndexError("Stack is empty")
        return self._items[-1]
    
    def is_empty(self):
        """判断是否为空"""
        return len(self._items) == 0
    
    def size(self):
        """栈的大小"""
        return len(self._items)
    
    def __repr__(self):
        return f"Stack({self._items})"


# 测试
stack = Stack()
stack.push(1)
stack.push(3)
stack.push(5)
print(stack)        # Stack([1, 3, 5])
print(stack.top())  # 5
print(stack.pop())  # 5
print(stack)        # Stack([1, 3])
```

### 2.4 栈的应用场景

```
1. 函数调用栈（递归调用）
2. 括号匹配
3. 表达式求值
4. 浏览器前进/后退
5. 撤销操作（Ctrl+Z）
```

### 2.5 经典应用：括号匹配

```python
def is_valid_parentheses(s: str) -> bool:
    """
    判断括号是否匹配
    示例: "()" -> True, "()[]{}" -> True, "(]" -> False
    """
    stack = []
    matching = {')': '(', ']': '[', '}': '{'}
    
    for char in s:
        if char in '([{':
            # 左括号入栈
            stack.append(char)
        else:
            # 右括号：检查是否匹配
            if not stack or stack[-1] != matching[char]:
                return False
            stack.pop()
    
    # 栈空说明全部匹配
    return len(stack) == 0


# 测试
print(is_valid_parentheses("()"))      # True
print(is_valid_parentheses("()[]{}"))  # True
print(is_valid_parentheses("(]"))      # False
print(is_valid_parentheses("([)]"))    # False
```

## 三、队列（Queue）

### 3.1 队列的定义

**专业解释**：队列是一种特殊的线性表，只允许在一端（队尾）插入，在另一端（队首）删除。

```
队列的结构：
    队首（front）                    队尾（rear）
       ↓                              ↓
    ┌─────────┬─────────┬─────────┬─────────┐
    │    1    │    3    │    5    │    7    │
    └─────────┴─────────┴─────────┴─────────┘
                                          ↑
                                     enqueue(9) 入队
    dequeue() 出队 ←
```

### 3.2 队列的基本操作

| 操作 | 说明 | 时间复杂度 |
|------|------|-----------|
| enqueue(x) | 入队，向队尾添加元素 | O(1) |
| dequeue() | 出队，移除队首元素 | O(1) |
| front() | 查看队首元素 | O(1) |
| rear() | 查看队尾元素 | O(1) |
| isEmpty() | 判断是否为空 | O(1) |
| size() | 返回队列大小 | O(1) |

### 3.3 队列的代码实现

```python
class Queue:
    """基于 Python list 的队列实现"""
    
    def __init__(self):
        self._items = []
    
    def enqueue(self, item):
        """入队"""
        self._items.append(item)
    
    def dequeue(self):
        """出队"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        return self._items.pop(0)  # 从头部移除
    
    def front(self):
        """查看队首"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        return self._items[0]
    
    def rear(self):
        """查看队尾"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        return self._items[-1]
    
    def is_empty(self):
        return len(self._items) == 0
    
    def size(self):
        return len(self._items)
    
    def __repr__(self):
        return f"Queue({self._items})"


# 测试
queue = Queue()
queue.enqueue(1)
queue.enqueue(3)
queue.enqueue(5)
print(queue)          # Queue([1, 3, 5])
print(queue.front())  # 1
print(queue.dequeue()) # 1
print(queue)          # Queue([3, 5])
```

### 3.4 队列的变体

```
普通队列：先进先出

循环队列：用数组实现，队首队尾相连
  ┌─────────────────┐
  ↓                 │
  ┌──┬──┬──┬──┬──┐   │
  │  │5 │  │1 │2 │───┘
  └──┴──┴──┴──┴──┘
    front=3, rear=0

双端队列（Deque）：两端都可以插入删除

优先队列：按优先级出队（用堆实现）
```

### 3.5 循环队列的实现

```python
class CircularQueue:
    """循环队列 - 用固定大小的数组实现"""
    
    def __init__(self, capacity):
        self.capacity = capacity
        self.queue = [None] * capacity
        self.front = 0   # 队首指针
        self.rear = 0    # 队尾指针
        self.size = 0
    
    def enqueue(self, item):
        """入队"""
        if self.is_full():
            raise OverflowError("Queue is full")
        self.queue[self.rear] = item
        self.rear = (self.rear + 1) % self.capacity
        self.size += 1
    
    def dequeue(self):
        """出队"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        item = self.queue[self.front]
        self.front = (self.front + 1) % self.capacity
        self.size -= 1
        return item
    
    def is_empty(self):
        return self.size == 0
    
    def is_full(self):
        return self.size == self.capacity
    
    def __repr__(self):
        if self.is_empty():
            return "CircularQueue([])"
        items = []
        i = self.front
        for _ in range(self.size):
            items.append(self.queue[i])
            i = (i + 1) % self.capacity
        return f"CircularQueue({items})"


# 测试
cq = CircularQueue(5)
for i in range(1, 6):
    cq.enqueue(i)
print(cq)           # CircularQueue([1, 2, 3, 4, 5])
print(cq.dequeue()) # 1
print(cq)           # CircularQueue([2, 3, 4, 5])
cq.enqueue(6)
print(cq)           # CircularQueue([2, 3, 4, 5, 6])
```

```
2. 广度优先搜索（BFS）
3. 缓存（最近请求）
4. 消息队列
5. 线程池任务分配
```

## 四、双端队列（Deque）

### 4.1 什么是双端队列？

**专业解释**：双端队列是一种两端都可以进行插入和删除操作的队列。

```
双端队列：
  ←── front                    rear ──→
┌─────────┬─────────┬─────────┬─────────┐
│    1    │    3    │    5    │    7    │
└─────────┴─────────┴─────────┴─────────┘
  ↑                              ↑
front_insert()              rear_insert()
front_remove()              rear_remove()
```

### 4.2 双端队列的代码实现

```python
from collections import deque

# Python 内置的 deque 就是双端队列
dq = deque()

dq.append(1)       # 右端添加
dq.appendleft(0)   # 左端添加
dq.append(3)

print(list(dq))    # [0, 1, 3]

dq.pop()           # 右端移除: 3
dq.popleft()       # 左端移除: 0
print(list(dq))    # [1]

# 常用方法
dq.append(2)
print(dq[0])       # 查看左端: 1
print(dq[-1])      # 查看右端: 2
```

### 4.3 双端队列的应用

```python
# 滑动窗口最大值
from collections import deque

def max_sliding_window(nums: list[int], k: int) -> list[int]:
    """
    找到滑动窗口中的最大值
    示例: nums=[1,3,-1,-3,5,3,6,7], k=3
    结果: [3,3,5,5,6,7]
    """
    if not nums or k == 0:
        return []
    
    result = []
    window = deque()  # 存储索引
    
    for i, num in enumerate(nums):
        # 移除超出窗口的元素
        while window and window[0] < i - k + 1:
            window.popleft()
        
        # 移除小于当前元素的索引（它们不可能是最大值）
        while window and nums[window[-1]] < num:
            window.pop()
        
        window.append(i)
        
        # 记录结果
        if i >= k - 1:
            result.append(nums[window[0]])
    
    return result


# 测试
print(max_sliding_window([1,3,-1,-3,5,3,6,7], 3))
# 输出: [3, 3, 5, 5, 6, 7]
```

## 五、栈与队列的对比

### 5.1 操作对比

| 特性 | 栈 | 队列 | 双端队列 |
|------|-----|------|---------|
| 入栈/入队 | push() | enqueue() | append() |
| 出栈/出队 | pop() | dequeue() | popleft() |
| 查看栈顶/队首 | top() | front() | [0] |
| 查看队尾 | - | rear() | [-1] |
| 顺序 | LIFO | FIFO | 两端都行 |

## 六、知识总结

### 6.1 一句话总结

```
栈 = 后悔药 = 后进先出 = 只开一个口
队列 = 排队 = 先进先出 = 两头开门
```

### 6.2 复杂度速查

| 操作 | 栈 | 队列 | 双端队列 |
|------|-----|------|---------|
| 头部插入 | - | O(1) | O(1) |
| 尾部插入 | O(1) | O(1) | O(1) |
| 头部删除 | - | O(1) | O(1) |
| 尾部删除 | O(1) | - | O(1) |
| 随机访问 | - | - | O(1) |

### 6.3 选择指南

```
需要"最近相关" → 用栈（函数调用、括号匹配、撤销）
需要"先来先服务" → 用队列（BFS、任务调度）
需要"两端操作" → 用双端队列（滑动窗口）
需要"优先级" → 用优先队列
```

## 七、自我检验

### 思考题

1. **基础题**：为什么递归可以用栈来理解？
2. **进阶题**：如何用两个栈实现一个队列？
3. **应用题**：设计一个"最小栈"，能 O(1) 获取最小值
4. **思考题**：浏览器的前进后退功能用什么数据结构实现？

### 参考答案

1. **递归和栈**：每次函数调用会把返回地址压栈，递归到最后从栈顶弹出依次返回。

2. **两个栈实现队列**：
   ```python
   class QueueWithTwoStacks:
       def __init__(self):
           self.stack_in = []
           self.stack_out = []
       
       def enqueue(self, x):
           self.stack_in.append(x)
       
       def dequeue(self):
           if not self.stack_out:
               while self.stack_in:
                   self.stack_out.append(self.stack_in.pop())
           return self.stack_out.pop()
   ```

3. **最小栈**：
   ```python
   class MinStack:
       def __init__(self):
           self.stack = []
           self.min_stack = []
       
       def push(self, x):
           self.stack.append(x)
           if not self.min_stack or x <= self.min_stack[-1]:
               self.min_stack.append(x)
       
       def pop(self):
           if self.stack.pop() == self.min_stack[-1]:
               self.min_stack.pop()
       
       def getMin(self):
           return self.min_stack[-1]
   ```

4. **浏览器前进后退**：用两个栈。一个栈存"后退"的页面，一个栈存"前进"的页面。
