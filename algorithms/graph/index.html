什么是图？

图由 顶点 (Vertex) 和 边 (Edge) 组成，记作 G = (V, E)，是描述 万物互联关系 的最通用数据结构。

🌐 无处不在：社交网络（人与人）、地图导航（地点与道路）、Web 页面（超链接）、神经网络（神经元连接）。

两种基本遍历方式：

  • BFS（广度优先）：层层推进，像水波扩散，找到最短路径
  • DFS（深度优先）：一路走到黑，回溯再走，遍历所有可能

💻 代码实现

from collections import deque

def bfs(graph, start):
    """graph: 邻接表 {node: [neighbors]}"""
    visited = set()
    queue = deque([start])
    visited.add(start)
    order = []

    while queue:
        node = queue.popleft()
        order.append(node)
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    return order
def dfs(graph, start):
    """递归版 DFS"""
    visited = set()
    order = []

    def dfs_recursive(node):
        visited.add(node)
        order.append(node)
        for neighbor in graph[node]:
            if neighbor not in visited:
                dfs_recursive(neighbor)

    dfs_recursive(start)
    return order

# 迭代版
def dfs_iterative(graph, start):
    visited = set()
    stack = [start]
    order = []
    while stack:
        node = stack.pop()
        if node not in visited:
            visited.add(node)
            order.append(node)
            # 逆序入栈保持顺序
            for n in reversed(graph[node]):
                if n not in visited:
                    stack.append(n)
    return order
