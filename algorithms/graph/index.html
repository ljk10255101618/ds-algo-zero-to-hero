图结构 Lv3

图 Graph

顶点 + 边，描述万物互联的关系

📝 定义

图由 顶点 (Vertex) 和 边 (Edge) 组成，记作 G = (V, E)。

分类：
  • 有向图 vs 无向图 — 边是否有方向
  • 加权图 vs 无权图 — 边是否带权重
  • 连通图 vs 非连通图 — 任意两点是否可达
存储方式： 邻接矩阵 O(V²) · 邻接表 O(V+E)
DFS O(V+E)
BFS O(V+E)
Dijkstra O(E log V)
空间 O(V+E)

⚡ 代码

from collections import deque

def bfs(graph, start):
    visited = {start}
    queue = deque([start])
    while queue:
        v = queue.popleft()
        for n in graph[v]:
            if n not in visited:
                visited.add(n)
                queue.append(n)
    return visited

def dfs(graph, start, visited=None):
    if visited is None:
        visited = set()
    visited.add(start)
    for n in graph[start]:
        if n not in visited:
            dfs(graph, n, visited)
    return visited
