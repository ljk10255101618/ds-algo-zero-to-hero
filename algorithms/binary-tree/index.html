什么是二叉树？

二叉树 是一种树形数据结构，其中每个节点最多有两个子节点：左子节点 和 右子节点。

🌳 想象一个 族谱：每个人最多有两个孩子。最上面的祖先叫 根节点，没有孩子的叫 叶子节点。从根到任意节点的层数叫 深度。

二叉树的 递归结构 是其核心特性：每个子树本身也是一棵二叉树。这使得许多树的操作（遍历、搜索）可以优雅地用递归实现。

🌲 遍历方式对比

遍历方式顺序用途
前序 (Pre-order)根 → 左 → 右复制树、序列化
中序 (In-order)左 → 根 → 右BST 有序输出
后序 (Post-order)左 → 右 → 根删除树、表达式求值
层序 (Level-order)逐层从左到右BFS、最短路径

🎯 特殊二叉树

  • 满二叉树：每个节点都有 0 或 2 个子节点
  • 完全二叉树：除最后一层外全满，叶子靠左排列（堆的底层结构）
  • 二叉搜索树 (BST)：左 < 根 < 右，中序遍历为递增序列
  • 平衡二叉树 (AVL)：左右子树高度差 ≤ 1
  • 完美二叉树：所有叶子在同一层，节点数 = 2^h - 1

💻 代码实现

Python二叉树遍历
class TreeNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

# 前序遍历：根 → 左 → 右
def preorder(root):
    if not root: return []
    return [root.val] + preorder(root.left) + preorder(root.right)

# 中序遍历：左 → 根 → 右
def inorder(root):
    if not root: return []
    return inorder(root.left) + [root.val] + inorder(root.right)

# 后序遍历：左 → 右 → 根
def postorder(root):
    if not root: return []
    return postorder(root.left) + postorder(root.right) + [root.val]

# 层序遍历 (BFS)
def levelorder(root):
    if not root: return []
    result, queue = [], [root]
    while queue:
        node = queue.pop(0)
        result.append(node.val)
        if node.left: queue.append(node.left)
        if node.right: queue.append(node.right)
    return result

🎯 关键要点

✅ 递归是树的灵魂 — 树的定义本身就是递归的，遍历和操作天然适合递归
✅ 四种遍历各有用途 — 中序→有序，前序→复制，后序→删除，层序→BFS
✅ 时间复杂度 — 所有遍历均为 O(n)，树的深度影响递归栈空间 O(h)

⏱️ 复杂度速查

操作复杂度
遍历O(n)
树高O(h)
递归栈O(h)

h 为树高, 最好 log n, 最差 n
