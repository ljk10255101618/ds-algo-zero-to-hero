双端队列 (Deque)

什么是双端队列？

双端队列（Deque） 是队列的升级版，允许在 头部和尾部两端 进行插入和删除操作。

想象火车站 🚉 的 双面站台 —— 乘客可以从左边上车，也可以从右边上车；可以从左边下车，也可以从右边下车。这就是 Deque 的 "双端" 特性。

在 Python 中，collections.deque 是高度优化的双端队列实现，提供了 O(1) 的头尾操作效率。

💻 代码实现

from collections import deque

d = deque()          # 创建空双端队列
d.append(1)          # push_back
d.appendleft(2)      # push_front
x = d.pop()          # pop_back
y = d.popleft()      # pop_front
d[0]                 # O(1) 随机访问（头部）
len(d)               # O(1) 获取大小
import java.util.ArrayDeque;

ArrayDeque<Integer> d = new ArrayDeque<>();
d.addLast(1);       // push_back
d.addFirst(2);      // push_front
int x = d.removeLast();   // pop_back
int y = d.removeFirst();  // pop_front
int front = d.peekFirst();// 查看头部
int size = d.size();
#include <deque>
std::deque<int> d;
d.push_back(1);     // push_back
d.push_front(2);    // push_front
d.pop_back();       // pop_back
d.pop_front();      // pop_front
int front = d.front();
int size = d.size();

📊 复杂度分析

操作时间复杂度说明
push_front / push_backO(1)两端插入，无需移动元素
pop_front / pop_backO(1)两端删除，无需移动元素
随机访问 (索引)O(1)分段连续存储，两级索引
中间插入/删除O(n)需要移动元素到两端
空间复杂度O(n)存储 n 个元素
