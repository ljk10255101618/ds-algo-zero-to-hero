队列 Queue

先进先出 (FIFO)，队尾入队、队头出队

📝 定义

队列是一种操作受限的线性表，只允许在队尾插入 (enqueue)，在队头删除 (dequeue)，遵循 先进先出 (FIFO) 原则。

核心操作：
  • enqueue(x) — 入队：在队尾插入 x，O(1)
  • dequeue() — 出队：移除队头元素并返回，O(1)
  • front() — 查看队头元素，O(1)
Enqueue O(1)
Dequeue O(1)
Front O(1)
空间 O(n)

⚡ 代码实现

from collections import deque

class Queue:
    def __init__(self):
        self.items = deque()

    def enqueue(self, val):
        self.items.append(val)

    def dequeue(self):
        return self.items.popleft()

    def front(self):
        return self.items[0]

    def is_empty(self):
        return len(self.items) == 0

    def size(self):
        return len(self.items)
