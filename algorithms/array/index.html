数组 Array

连续内存存储，索引访问高效的基础数据结构

📝 定义

数组是最基本的数据结构，用一段连续的内存空间存储相同类型的元素。通过索引可以在 O(1) 时间内随机访问任意元素。

核心特点：连续的内存空间，支持 O(1) 的随机访问，但插入/删除操作需要移动元素。
访问 O(1)
搜索 O(n)
插入 O(n)
删除 O(n)

⚡ 代码实现

# Python 列表操作
arr = [1, 2, 3, 4, 5]

# 访问 O(1)
item = arr[2]  # 3

# 尾部操作 O(1)
arr.append(6)   # [1,2,3,4,5,6]
arr.pop()       # [1,2,3,4,5]

# 头部操作 O(n)
arr.insert(0, 0)  # [0,1,2,3,4,5]
arr.pop(0)        # [1,2,3,4,5]
