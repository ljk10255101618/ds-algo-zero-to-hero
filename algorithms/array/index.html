数组 (Array)

什么是数组？

数组是最基础、最常用的数据结构。 它在内存中占据一块连续的存储空间，用于存储相同类型的元素集合。

想象一栋公寓楼的 信箱墙 📮 —— 每个信箱有固定的编号（索引 0, 1, 2...），你知道 7 号信箱的位置，就能直接走过去打开它。这就是数组的 O(1) 随机访问。

但如果你要在中间插入一个新信箱，就需要把后面的所有信箱都挪动位置 —— 这就是数组插入需要 O(n) 时间 的原因。

📚 深度理解

🎯 适用场景

  • 需要频繁随机访问：通过索引快速获取元素
  • 数据量相对固定：不频繁进行插入/删除操作
  • 缓存友好：连续内存空间，CPU 缓存预取效率高
  • 矩阵运算：二维数组是矩阵的天然表示
  • 作为底层构建块：栈、队列、哈希表等都可以用数组实现

⏱️ 复杂度分析

操作时间复杂度说明
随机访问O(1)通过索引直接计算地址
搜索（无序）O(n)需要遍历查找
搜索（有序）O(log n)配合二分搜索
插入（尾部）O(1)有空间时直接写入
插入（中间/头部）O(n)需要移动后续元素
删除O(n)需要移动后续元素

💡 数组的 O(1) 随机访问 是其最大优势——地址计算：base_addr + index × element_size

⚠️ 局限性

  • 固定大小：创建后无法动态扩容（需要重新分配）
  • 插入/删除开销大：需要移动大量元素
  • 内存碎片：大数组需要连续大块内存
  • 类型单一：所有元素必须是相同类型

💻 代码实现

Python 数组基本操作 
# Python 列表模拟数组操作

# 创建数组
arr = [1, 3, 5, 7, 9]

# 随机访问 — O(1)
element = arr[2]     # → 5

# 搜索 — O(n)
def search(arr, target):
    for i, val in enumerate(arr):
        if val == target:
            return i
    return -1

# 插入 — O(n)
def insert(arr, index, value):
    arr.append(0)  # 先扩展
    # 从后往前移动元素
    for i in range(len(arr)-1, index, -1):
        arr[i] = arr[i-1]
    arr[index] = value
    return arr

# 删除 — O(n)
def delete(arr, index):
    for i in range(index, len(arr)-1):
        arr[i] = arr[i+1]
    arr.pop()
    return arr

🎯 关键要点

✅ O(1) 随机访问 是数组最大的优势，源于连续内存和索引计算
✅ 缓存友好 — 连续内存使得 CPU 可以预加载相邻数据，访问速度远快于链表
⚠️ 插入/删除 O(n) — 需要移动元素，大量操作时考虑使用链表
💡 双指针技巧 是数组相关算法中最常用的优化手段
