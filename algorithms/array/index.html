数组 Array

连续内存空间，O(1) 随机访问

📝 定义

数组是最基础、最常用的数据结构。它在内存中占据一块连续的存储空间，用于存储相同类型的元素集合。

核心特点：
  • 连续内存 — 数组元素在内存中紧紧相邻存放
  • 索引访问 — 通过索引直接计算地址：addr[i] = base + i × size
  • 固定类型 — 所有元素必须是相同的数据类型
访问 O(1)
搜索 O(n)
插入 O(n)
删除 O(n)

💡 适用场景

🎯 频繁随机访问

通过索引快速获取元素，O(1) 效率

💾 缓存友好

连续内存空间，CPU 缓存预取效率高

📐 矩阵运算

二维数组是矩阵的天然表示

⚠️ 局限性

  • 固定大小：创建后无法动态扩容（需要重新分配）
  • 插入/删除开销大：需要移动大量元素
  • 内存碎片：大数组需要连续大块内存

⚡ 代码实现

# Python 数组操作

# 创建数组
arr = [1, 3, 5, 7, 9]

# 随机访问 — O(1)
element = arr[2]     # → 5

# 搜索 — O(n)
def search(arr, target):
    for i, val in enumerate(arr):
        if val == target:
            return i
    return -1

# 插入 — O(n)
def insert(arr, idx, val):
    arr.append(0)  # 先扩展
    for i in range(len(arr)-1, idx, -1):
        arr[i] = arr[i-1]
    arr[idx] = val

# 删除 — O(n)  
def delete(arr, idx):
    for i in range(idx, len(arr)-1):
        arr[i] = arr[i+1]
    arr.pop()
