什么是树状数组？

树状数组 (Binary Indexed Tree, BIT) 是一种用于 高效维护前缀和 的数据结构，由 Peter Fenwick 在 1994 年提出。

它的核心思想是利用 lowbit（二进制最低位的 1）将数组组织成树形结构，实现 O(log n) 的更新和查询。

💡 与线段树的对比：树状数组代码更简洁、常数更小，但只能处理前缀/差分相关的操作（求和、求逆序对等）。

💻 代码实现

class FenwickTree:
    def __init__(self, n):
        self.n = n
        self.bit = [0] * (n + 1)  # 1-indexed

    def lowbit(self, x):
        return x & -x

    def add(self, idx, delta):
        """idx 处增加 delta (1-indexed)"""
        i = idx
        while i <= self.n:
            self.bit[i] += delta
            i += self.lowbit(i)

    def prefix_sum(self, idx):
        """前缀和 [1, idx]"""
        res = 0
        i = idx
        while i > 0:
            res += self.bit[i]
            i -= self.lowbit(i)
        return res

    def range_sum(self, l, r):
        """区间和 [l, r]"""
        return self.prefix_sum(r) - self.prefix_sum(l - 1)

# 应用：求逆序对
def count_inversions(arr):
    bit = FenwickTree(max(arr))
    inv = 0
    for i, v in enumerate(arr):
        bit.add(v, 1)
        inv += i + 1 - bit.prefix_sum(v)
    return inv

📊 对比线段树

特性树状数组线段树
代码量少（5行核心）多
常数小较大
功能前缀和/差分任意区间操作
区间更新需差分技巧原生支持 lazy

📖 lowbit 详解

  • lowbit(x) = x & -x
  • 例: lowbit(6)=2 (110→10)
  • 更新时向上累加
  • 查询时向下减
