栈 Stack

后进先出 (LIFO)，只有一端可操作

📝 定义

栈是一种受限的线性表，只允许在表的一端（栈顶） 进行插入和删除操作，遵循 后进先出 (LIFO) 原则。

核心操作：
  • push(x) — 压栈：将 x 放入栈顶，O(1)
  • pop() — 出栈：移除并返回栈顶元素，O(1)
  • peek() — 查看栈顶元素，O(1)
Push O(1)
Pop O(1)
Peek O(1)
空间 O(n)

⚡ 代码实现

class Stack:
    def __init__(self):
        self.items = []

    def push(self, val):
        self.items.append(val)

    def pop(self):
        return self.items.pop()

    def peek(self):
        return self.items[-1]

    def is_empty(self):
        return len(self.items) == 0

    def size(self):
        return len(self.items)
