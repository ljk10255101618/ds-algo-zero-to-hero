链表 Linked List

动态数据结构的基石，插入删除 O(1)，随机访问 O(n)

📝 定义

链表是一种线性数据结构，元素通过指针链接而非连续存储。每个节点包含 数据域 和 指针域。

核心特性：
  • 动态大小，无需预分配内存
  • 插入/删除操作 O(1)（已知位置）
  • 随机访问 O(n)，不支持下标索引
  • 额外存储指针开销
访问 O(n)
搜索 O(n)
插入 O(1)
删除 O(1)

⚡ 代码实现

class ListNode:
    def __init__(self, val=0):
        self.val = val
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def insert_head(self, val):
        node = ListNode(val)
        node.next = self.head
        self.head = node

    def delete(self, val):
        curr, prev = self.head, None
        while curr:
            if curr.val == val:
                if prev: prev.next = curr.next
                else: self.head = curr.next
                return
            prev, curr = curr, curr.next
