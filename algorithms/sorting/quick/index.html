快速排序 (Quick Sort)

什么是快速排序？

快速排序 是应用最广泛的排序算法之一，它的核心思想是 分治法 (Divide and Conquer)。

🧩 想象你有一副 乱序的扑克牌：随便抽一张作为"基准"（pivot），比它小的放左边，比它大的放右边，然后对左右两堆重复这个过程。最终整副牌就有序了！

快速排序的 平均时间复杂度 O(n log n)，且是 原地排序（空间复杂度 O(log n)），这使得它在实践中比归并排序更快。

📚 深度理解

📝 算法步骤

  1. 选择 pivot：从数组中选一个元素作为基准
  2. 分区 (Partition)：小于 pivot 的放左边，大于的放右边
  3. 递归：对左右两个子数组重复步骤 1-2
  4. 终止：子数组长度 ≤ 1 时停止

💻 代码实现

Python快速排序实现
def quick_sort(arr, low, high):
    if low < high:
        # 分区，返回 pivot 的最终位置
        pi = partition(arr, low, high)
        # 递归排序左右两部分
        quick_sort(arr, low, pi - 1)
        quick_sort(arr, pi + 1, high)

def partition(arr, low, high):
    # 选最右边的元素作为 pivot
    pivot = arr[high]
    # i 指向小于 pivot 区域的边界
    i = low - 1
    
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    
    # 将 pivot 放到正确位置
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

⏱️ 复杂度分析

情况时间复杂度说明
最好O(n log n)每次 pivot 都在中间
平均O(n log n)随机数据下期望复杂度
最差O(n²)有序数组且选第一个/最后一个为 pivot

💡 优化方案：随机选 pivot、三数取中法、小数组切换插入排序

⚔️ 快排 vs 归并排序

特性快速排序归并排序
稳定性❌ 不稳定✅ 稳定
空间复杂度O(log n) 原地O(n) 额外数组
最差情况O(n²)O(n log n)
缓存友好✅ 好❌ 一般
实际速度⚡ 通常更快稳定但略慢

🎯 关键要点

✅ 分治思想 — 将大问题分解为独立的小问题分别解决
✅ 原地排序 — 不需要额外数组，空间效率高
⚠️ 不稳定 — 相等的元素可能会交换位置
💡 Pivot 选择 是关键 — 好的 pivot 选择能避免最差情况
