冒泡排序 Bubble Sort

相邻元素两两比较，大的"冒泡"到末尾

📝 定义

冒泡排序重复遍历数组，依次比较相邻两个元素，如果顺序错误就交换，直到没有需要交换的元素。

优化技巧：如果一趟遍历没有发生交换，说明已经有序，提前终止。
最优 O(n)
最差 O(n²)
平均 O(n²)
空间 O(1)

⚡ 代码实现

def bubble_sort(arr):
    n = len(arr)
    for i in range(n-1):
        swapped = False
        for j in range(n-1-i):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr
