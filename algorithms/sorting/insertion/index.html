什么是插入排序？

插入排序 的原理类似于 整理手中的扑克牌 🃏 —— 每次从桌上拿起一张新牌，插入到手中已排好序的合适位置。

它的核心思想是：将待排序元素插入到已排序序列的正确位置。

特点：稳定排序，对近乎有序的数据效率极高（O(n)），是高级排序算法（如 Timsort）的重要组成部分。

💻 代码实现

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]  # 向右移动
            j -= 1
        arr[j + 1] = key  # 插入到正确位置
    return arr

情况时间复杂度空间
最好（已有序）O(n)O(1)
平均O(n²)
最坏（逆序）O(n²)

  • 稳定排序 ✅
