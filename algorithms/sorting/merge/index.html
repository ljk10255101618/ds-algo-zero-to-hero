归并排序 Merge Sort

分治思想的经典代表，稳定排序，O(n log n)

📝 定义

归并排序采用 分治法 (Divide and Conquer)：将数组分成两半分别排序，再合并两个有序子数组。

算法步骤：
  1. 分 (Divide)：将数组从中间切分为左右两半
  2. 治 (Conquer)：递归地对两半分别排序
  3. 合 (Merge)：将两个有序子数组合并为一个有序数组
最优 O(n log n)
最差 O(n log n)
平均 O(n log n)
空间 O(n)

⚡ 代码

def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    res, i, j = [], 0, 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            res.append(left[i]); i += 1
        else:
            res.append(right[j]); j += 1
    return res + left[i:] + right[j:]
