归并排序 Merge Sort

分治思想，先拆分再合并，稳定高效

📝 定义

归并排序采用分治策略，将数组递归分成两半，分别排序后再合并成一个有序数组。

核心思想：分而治之。将数组不断二分，直到每个子数组只有一个元素，然后逐步合并。
最优 O(n log n)
最差 O(n log n)
平均 O(n log n)
空间 O(n)

def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
