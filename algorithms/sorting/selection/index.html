选择排序 (Selection Sort)

什么是选择排序？

选择排序 是一种直观的排序算法。它的核心思想是：每一轮从未排序部分选出最小值，放到已排序部分的末尾。

📱 就像你手里有一副打乱的扑克牌，你每次从中抽出最小的那张放到桌上 —— 重复这个操作，桌上的牌就是有序的。

特点：不稳定排序，原地排序，无论数据如何，比较次数都相同。

💻 代码实现

def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]  # 交换
    return arr

情况时间复杂度空间

📖 特性

  • 原地排序 O(1)
  • 比较次数固定 n(n-1)/2
