线段树 (Segment Tree)

什么是线段树？

线段树 是一种用于 区间查询和区间更新 的二叉树数据结构。它可以在 O(log n) 时间内完成区间求和、区间最小值/最大值等操作。

🌳 线段树将一个大区间递归地分成两半，每个节点代表一个区间 [l, r]，叶子节点代表单个元素。

💡 典型应用：区间和查询 + 单点/区间更新、区间最大值/最小值、区间众数。

💻 代码实现

class SegmentTree:
    def __init__(self, arr):
        n = len(arr)
        self.tree = [0] * (4 * n)
        self.arr = arr
        self._build(0, 0, n - 1)

    def _build(self, node, l, r):
        if l == r:
            self.tree[node] = self.arr[l]
            return
        mid = (l + r) // 2
        self._build(node * 2 + 1, l, mid)
        self._build(node * 2 + 2, mid + 1, r)
        self.tree[node] = self.tree[node * 2 + 1] + self.tree[node * 2 + 2]

    def update(self, idx, val, node=0, l=0, r=None):
        if r is None: r = len(self.arr) - 1
        if l == r:
            self.tree[node] = val
            self.arr[idx] = val
            return
        mid = (l + r) // 2
        if idx <= mid:
            self.update(idx, val, node * 2 + 1, l, mid)
        else:
            self.update(idx, val, node * 2 + 2, mid + 1, r)
        self.tree[node] = self.tree[node * 2 + 1] + self.tree[node * 2 + 2]

    def query(self, ql, qr, node=0, l=0, r=None):
        if r is None: r = len(self.arr) - 1
        if ql <= l and r <= qr: return self.tree[node]
        if r < ql or l > qr: return 0
        mid = (l + r) // 2
        return self.query(ql, qr, node * 2 + 1, l, mid) + \
               self.query(ql, qr, node * 2 + 2, mid + 1, r)

📊 复杂度

操作时间复杂度空间
建树O(n)O(4n)
单点更新O(log n)
区间查询O(log n)

📖 核心操作

  • build: 递归建树 O(n)
  • update: 更新叶子 + 回溯 O(log n)
  • query: 区间合并 O(log n)
  • lazy propagation: 区间更新
