堆 Heap

完全二叉树 + 堆序性质，优先队列的核心

📝 定义

堆是一种 完全二叉树，满足 堆序性质：

  • 大根堆 (Max-Heap)：父节点 ≥ 子节点，根是最大值
  • 小根堆 (Min-Heap)：父节点 ≤ 子节点，根是最小值
插入 O(log n)
取极值 O(1)
删除极值 O(log n)
建堆 O(n)

🎮 交互演示

💡 应用

⏰ 任务调度

操作系统按优先级调度进程

📊 Top K

小根堆维护最大的 K 个元素

🔀 堆排序

建堆 O(n) + 取极值 O(n log n)

⚡ 代码

import heapq
# Python 默认小根堆
heap = []
heapq.heappush(heap, 3)
heapq.heappush(heap, 1)
heapq.heappush(heap, 2)
min_val = heapq.heappop(heap)  # 1

# 大根堆：存负值
import heapq
max_heap = []
heapq.heappush(max_heap, -3)
heapq.heappush(max_heap, -1)
max_val = -heapq.heappop(max_heap)  # 3
