## 一、什么是动态规划？

### 1.1 从生活中的例子理解

```
爬楼梯问题：
有 10 级台阶，每次可以爬 1 或 2 级，问有多少种爬法？

方法1（暴力）：枚举所有可能 → 2^10 种组合 ❌

方法2（找规律）：
  - 第 10 级 = 第 9 级爬 1 步 + 第 8 级爬 2 步
  - f(10) = f(9) + f(8)
  - f(9) = f(8) + f(7)
  - ...
  
  递推公式：f(n) = f(n-1) + f(n-2)
  
  f(1) = 1, f(2) = 2
  f(3) = 3, f(4) = 5, f(5) = 8, ... f(10) = 89 ✅
```

### 1.2 动态规划的核心思想

**化繁为简**：把大问题分解成小问题，小问题的解可以重用。

```
斐波那契数列：
  普通递归：                        动态规划：
       fib(5)                         fib(5)
      /      \                      /      \
  fib(4)    fib(3)                 fib(4)  fib(3)
  /    \    /    \               /    \  /    \
fib(3) fib(2) fib(2) fib(1)    fib(3) fib(2) fib(2) fib(1)
  ...重复计算很多！                 记录结果，不重复计算！
```

**三个关键要素**：
1. **最优子结构**：大问题可以由小问题的最优解构成
2. **重叠子问题**：小问题会被重复计算
3. **状态转移**：如何从子问题推导到大问题

> [!tip] 核心问题
> - 什么情况下可以用动态规划？
> - 如何找到状态转移方程？
> - 自顶向下（递归+记忆化）vs 自底向上（迭代）？

## 二、动态规划入门：斐波那契

### 2.1 普通递归（效率低）

```python
def fib_recursive(n):
    """普通递归 - 有大量重复计算"""
    if n <= 1:
        return n
    return fib_recursive(n - 1) + fib_recursive(n - 2)

# 问题：fib(5) 要计算 fib(4) 和 fib(3)
# fib(4) 要计算 fib(3) 和 fib(2)
# fib(3) 被计算了 2 次！
# fib(3) 下面还有更多重复...
```

### 2.2 带记忆化的递归（自顶向下）

```python
from functools import lru_cache

@lru_cache(maxsize=None)
def fib_memoized(n):
    """记忆化递归 - 避免重复计算"""
    if n <= 1:
        return n
    return fib_memoized(n - 1) + fib_memoized(n - 2)


def fib_memo_manual(n, memo=None):
    """手动实现记忆化"""
    if memo is None:
        memo = {}
    
    if n in memo:
        return memo[n]
    if n <= 1:
        return n
    
    memo[n] = fib_memo_manual(n - 1, memo) + fib_memo_manual(n - 2, memo)
    return memo[n]


# 测试
import time

n = 40

start = time.time()
result1 = fib_recursive(n)
print(f"普通递归: {time.time() - start:.4f}s")  # 非常慢！

start = time.time()
result2 = fib_memoized(n)
print(f"记忆化: {time.time() - start:.4f}s")    # 非常快！

print(f"结果: {result1}, {result2}")  # 都是 102334155
```

### 2.3 迭代实现（自底向上）

```python
def fib_iterative(n):
    """迭代实现 - 空间优化"""
    if n <= 1:
        return n
    
    prev, curr = 0, 1
    for _ in range(2, n + 1):
        prev, curr = curr, prev + curr
    return curr


def fib_dp_array(n):
    """动态规划 - 用数组保存所有状态"""
    if n <= 1:
        return n
    
    dp = [0] * (n + 1)
    dp[0], dp[1] = 0, 1
    
    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    
    return dp[n]


# 测试
print(fib_iterative(10))   # 55
print(fib_dp_array(10))    # 55
```

## 三、经典动态规划问题

### 3.1 爬楼梯问题

```python
def climb_stairs(n):
    """
    爬 n 级楼梯，每次爬 1 或 2 级，有多少种方法？
    
    递推：f(n) = f(n-1) + f(n-2)
    f(1) = 1, f(2) = 2
    """
    if n <= 2:
        return n
    
    prev, curr = 1, 2
    for _ in range(3, n + 1):
        prev, curr = curr, prev + curr
    return curr


def climb_stairs_with_cost(n, cost):
    """
    爬楼梯，每次爬 1 或 2 级，每步有不同的体力消耗
    问最少消耗多少？
    """
    if n <= 1:
        return cost[0] if n == 0 else min(cost[0], cost[1])
    
    dp = [0] * n
    dp[0] = cost[0]
    dp[1] = cost[1]
    
    for i in range(2, n):
        dp[i] = min(dp[i - 1], dp[i - 2]) + cost[i]
    
    return dp[-1]


# 变体：可以爬 1, 2, 3 级
def climb_stairs_k(n, k=3):
    """可以爬 1 到 k 级"""
    if n <= 1:
        return n
    
    dp = [0] * (n + 1)
    dp[0], dp[1] = 1, 1
    
    for i in range(2, n + 1):
        for j in range(1, k + 1):
            if i >= j:
                dp[i] += dp[i - j]
    
    return dp[n]
```

### 3.2 零钱兑换问题

```python
def coin_change(coins, amount):
    """
    用最少的硬币凑成 amount
    coins = [1, 2, 5], amount = 11
    最少 3 枚：5 + 5 + 1 = 11
    
    dp[i] = 凑成 i 元需要的最少硬币数
    dp[i] = min(dp[i - coin] + 1) for all coins
    """
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    
    for i in range(1, amount + 1):
        for coin in coins:
            if coin <= i:
                dp[i] = min(dp[i], dp[i - coin] + 1)
    
    return dp[amount] if dp[amount] != float('inf') else -1


def coin_change_2(coins, amount):
    """求凑成 amount 的方法数"""
    dp = [0] * (amount + 1)
    dp[0] = 1
    
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] += dp[i - coin]
    
    return dp[amount]


def coin_change_ways(coins, amount):
    """完全背包问题：凑成 amount 的组合数（顺序无关）"""
    dp = [0] * (amount + 1)
    dp[0] = 1
    
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] += dp[i - coin]
    
    return dp[amount]


# 测试
coins = [1, 2, 5]
print(coin_change(coins, 11))     # 3
print(coin_change_2(coins, 11))   # 11
```

### 3.3 打家劫舍问题

```python
def rob(nums):
    """
    不能偷相邻的两家，最多能偷多少钱
    nums = [1, 2, 3, 1]
    偷 1 + 3 = 4
    
    dp[i] = max(dp[i-1], dp[i-2] + nums[i])
    """
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    
    prev, curr = nums[0], max(nums[0], nums[1])
    
    for i in range(2, len(nums)):
        prev, curr = curr, max(curr, prev + nums[i])
    
    return curr


def rob_circle(nums):
    """环形数组版本：第一家和最后一家不能同时偷"""
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    
    def rob_linear(nums):
        prev, curr = 0, 0
        for num in nums:
            prev, curr = curr, max(curr, prev + num)
        return curr
    
    # 情况1：不偷最后一家
    return max(rob_linear(nums[:-1]), rob_linear(nums[1:]))


def rob_tree(root):
    """二叉树版本：不能偷相邻节点"""
    def dfs(node):
        if not node:
            return (0, 0)
        
        left = dfs(node.left)
        right = dfs(node.right)
        
        # 不偷当前节点 = max(偷左子树) + max(偷右子树)
        # 偷当前节点 = 当前值 + 不偷左子树 + 不偷右子树
        not_take = max(left) + max(right)
        take = node.val + left[0] + right[0]
        
        return (not_take, take)
    
    return max(dfs(root))
```

### 3.4 最长递增子序列

```python
def length_of_lis(nums):
    """
    最长递增子序列
    nums = [10, 9, 2, 5, 3, 7, 101, 18]
    LIS = [2, 3, 7, 101]，长度 4
    
    dp[i] = 以 nums[i] 结尾的 LIS 长度
    dp[i] = max(dp[j] + 1) for j < i and nums[j] < nums[i]
    """
    if not nums:
        return 0
    
    dp = [1] * len(nums)
    
    for i in range(1, len(nums)):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    
    return max(dp)


def length_of_lis_binary(nums):
    """二分优化版本 O(n log n)"""
    import bisect
    
    tails = []
    
    for num in nums:
        # 找到第一个 >= num 的位置
        pos = bisect.bisect_left(tails, num)
        
        if pos == len(tails):
            tails.append(num)
        else:
            tails[pos] = num
    
    return len(tails)


def reconstruct_lis(nums):
    """返回实际的 LIS"""
    if not nums:
        return []
    
    # dp[i] = 以 nums[i] 结尾的 LIS 长度和路径
    dp_len = [1] * len(nums)
    dp_path = [[num] for num in nums]
    
    for i in range(1, len(nums)):
        for j in range(i):
            if nums[j] < nums[i] and dp_len[j] + 1 > dp_len[i]:
                dp_len[i] = dp_len[j] + 1
                dp_path[i] = dp_path[j] + [nums[i]]
    
    # 找到最长的
    max_idx = max(range(len(dp_len)), key=lambda i: dp_len[i])
    return dp_path[max_idx]


# 测试
nums = [10, 9, 2, 5, 3, 7, 101, 18]
print(length_of_lis(nums))           # 4
print(length_of_lis_binary(nums))    # 4
print(reconstruct_lis(nums))        # [2, 3, 7, 18]
```

### 3.5 最长公共子序列

```python
def longest_common_subsequence(text1, text2):
    """
    最长公共子序列
    text1 = "abcde", text2 = "ace"
    LCS = "ace"，长度 3
    
    dp[i][j] = text1[:i] 和 text2[:j] 的 LCS 长度
    if text1[i-1] == text2[j-1]: dp[i][j] = dp[i-1][j-1] + 1
    else: dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    """
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    return dp[m][n]


def lcs_with_path(text1, text2):
    """返回 LCS 的实际字符串"""
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    # 回溯找 LCS
    i, j = m, n
    lcs = []
    while i > 0 and j > 0:
        if text1[i - 1] == text2[j - 1]:
            lcs.append(text1[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            i -= 1
        else:
            j -= 1
    
    return ''.join(reversed(lcs))


# 测试
text1 = "abcde"
text2 = "ace"
print(longest_common_subsequence(text1, text2))  # 3
print(lcs_with_path(text1, text2))                 # "ace"
```

### 3.6 背包问题

```python
def knapsack_01(weights, values, capacity):
    """
    0-1 背包问题
    每个物品只能选一次
    
    dp[i][w] = 前 i 个物品，容量为 w 的最大价值
    dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight[i]] + value[i])
    """
    n = len(weights)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]
    
    for i in range(1, n + 1):
        for w in range(capacity + 1):
            if weights[i - 1] <= w:
                dp[i][w] = max(dp[i - 1][w], 
                              dp[i - 1][w - weights[i - 1]] + values[i - 1])
            else:
                dp[i][w] = dp[i - 1][w]
    
    return dp[n][capacity]


def knapsack_01_optimized(weights, values, capacity):
    """空间优化版本"""
    dp = [0] * (capacity + 1)
    
    for i in range(len(weights)):
        for w in range(capacity, weights[i] - 1, -1):  # 逆序！
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    
    return dp[capacity]


def knapsack_complete(weights, values, capacity):
    """完全背包：每种物品无限个"""
    dp = [0] * (capacity + 1)
    
    for i in range(len(weights)):
        for w in range(weights[i], capacity + 1):  # 正序！
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    
    return dp[capacity]


def knapsack_multi(weights, values, nums, capacity):
    """多重背包：每种物品有限个"""
    dp = [0] * (capacity + 1)
    
    for i in range(len(weights)):
        for w in range(capacity + 1):
            for k in range(nums[i] + 1):
                if w >= k * weights[i]:
                    dp[w] = max(dp[w], dp[w - k * weights[i]] + k * values[i])
    
    return dp[capacity]


# 测试
weights = [2, 3, 4, 5]
values = [3, 4, 5, 6]
capacity = 8
print(knapsack_01(weights, values, capacity))      # 10
print(knapsack_01_optimized(weights, values, capacity))  # 10
```

## 四、动态规划模板

### 4.1 解题步骤

```
1. 确定状态
   - 问题问什么，状态就表示什么
   - 往往是最优解或计数

2. 确定状态转移方程
   - dp[i] = ?
   - dp[i][j] = ?
   
3. 确定初始状态
   - dp[0] = ?
   - dp[...][0] = ?

4. 确定计算顺序
   - 从小到大
   - 从前往后
```

## 五、知识总结

### 5.1 一句话总结

```
动态规划 = 分治 + 记忆化
          = 用空间换时间
          = 找规律，建方程
```

### 5.2 常见 DP 模型

| 模型 | 状态定义 | 转移方程 |
|------|---------|---------|
| 线性 DP | dp[i] | dp[i] = f(dp[0..i-1]) |
| 区间 DP | dp[i][j] | dp[i][j] = f(dp[i+1][j], dp[i][j-1]) |
| 背包 DP | dp[i][w] | dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight]+value) |
| 树形 DP | dp[node] | dp[node] = f(dp[children]) |
| 状态压缩 | dp[mask] | dp[mask] = min(dp[mask without i] + cost[i]) |

### 5.3 空间优化技巧

```
- 数组变滚动数组
- 二维变一维
- 只保留必要的状态
```

## 六、自我检验

### 思考题

1. **基础题**：动态规划和分治有什么区别？
2. **进阶题**：如何判断一个问题能不能用动态规划？
3. **应用题**：编辑距离（将 "horse" 变成 "ros" 最少需要几步？）
4. **思考题**：动态规划的时间复杂度如何分析？

### 参考答案

1. **动态规划 vs 分治**：
   - 分治：子问题相互独立
   - DP：子问题有重叠，且有最优子结构

2. **判断能否用 DP**：
   - 有最优子结构
   - 有重叠子问题
   - 可以定义状态和转移

3. **编辑距离**：
   ```python
   def min_distance(word1, word2):
       m, n = len(word1), len(word2)
       dp = [[0] * (n + 1) for _ in range(m + 1)]
       
       for i in range(m + 1):
           dp[i][0] = i
       for j in range(n + 1):
           dp[0][j] = j
       
       for i in range(1, m + 1):
           for j in range(1, n + 1):
               if word1[i-1] == word2[j-1]:
                   dp[i][j] = dp[i-1][j-1]
               else:
                   dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
       
       return dp[m][n]
   ```

4. **时间复杂度分析**：O(状态数 × 转移数)
